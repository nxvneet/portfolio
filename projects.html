Date of Birth Prime Checker

The "Date of Birth Prime Checker" is a simple web application that allows users to check if their date of birth is a "prime date." The user provides their name and date of birth (in the format DDMM) through an input form. The application then calculates whether the day and month of the given date are prime numbers or not.
